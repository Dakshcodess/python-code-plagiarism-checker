# Python Code Plagiarism Checker

## Overview
A desktop GUI application that detects plagiarism between Python source code files.

## Features
- Upload two Python files
- Hybrid plagiarism scoring
- Risk level detection
- View matched suspicious lines
- Save history using SQLite
- Export text reports

## Technologies Used
- Python
- Tkinter
- SQLite
- scikit-learn

## Run Project

python main.py
